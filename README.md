# React Frontend UI Portfolio

This repository contains my personal **UI / Frontend portfolio** built with **React.js**.
The project follows a clean, scalable, and production-ready frontend structure,
focused on reusable components and maintainable styling.

---

## Overview

- Component-driven UI development with React
- Custom SASS (SCSS) architecture
- No default or framework CSS usage
- Clean separation between global styles and component styles

---

## Tech Stack

- React.js (Vite)
- JavaScript (ES6+)
- SASS / SCSS
- HTML5

---

## Styling Architecture

- Custom CSS reset and global styles
- Centralized variables, mixins, and functions
- Component-level SCSS colocated with components
- Single SASS entry point for predictable styling flow

---

## Project Intent

- Build reusable and consistent UI components
- Maintain clean and readable frontend code
- Follow scalable styling and folder architecture
- Serve as a long-term frontend/UI reference project

---

## Setup

npm install
npm run dev
